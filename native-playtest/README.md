# Volleyball Dynasty Native Playtest

This is a separate React Native (Expo) playtest app focused on mobile-first UX:

- Position-by-position tryouts with quota validation
- Autosave to local device storage after every action
- Drag-and-drop lineup editor (6 court slots + libero)
- In-season JV -> Varsity call-ups (max 3, no send-downs)

## Run

1. `cd native-playtest`
2. `npm install`
3. `npm start`
4. Open in Expo Go (iOS/Android) or simulator

## Notes

- Save key: `vd_native_playtest_v1`
- This is intentionally a playtest vertical slice, not full feature parity with the web version.
